# YouTube Navigation

## Search
- Search input: `input#search`
- Search button: `button#search-icon-legacy`
- Voice search available via microphone icon

## Video Player
- Play/Pause: click video or press `k`
- Fullscreen: `f` key or button
- Mute: `m` key
- Skip: `j` (back 10s), `l` (forward 10s)
- Speed: Settings gear â†’ Playback speed

## Video Page
- Subscribe button below video
- Like/Dislike buttons
- Share button for link
- Description expandable below title
- Comments section below description

## Navigation
- Home: YouTube logo or Home in sidebar
- Subscriptions: sidebar menu
- Library: sidebar for history, playlists
- Shorts: vertical short-form videos

## Tips
- Add `&t=120` to URL to start at 2 minutes
- Use `?v=VIDEO_ID` format for direct links
