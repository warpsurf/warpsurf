# GitHub Navigation

## Keyboard Shortcuts
- Press `/` to focus search
- Press `t` for file finder in repos
- Press `g c` to go to Code tab
- Press `g i` to go to Issues
- Press `g p` to go to Pull Requests

## Repository Navigation
- Code tab shows files and README
- Issues tab for bug reports and features
- Pull requests tab for code reviews
- Actions tab for CI/CD workflows

## Search
- Global search bar at top
- Use `repo:owner/name` to scope search
- Use `is:issue` or `is:pr` to filter type
- Use `is:open` or `is:closed` for status

## Common Actions
- Fork button at top right of repo
- Star button next to fork
- Clone URL in green "Code" dropdown
- New issue: Issues tab → "New issue" button
- New PR: Pull requests tab → "New pull request"
