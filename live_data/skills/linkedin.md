# LinkedIn Navigation

## Search
- Search bar at top: `.search-global-typeahead__input`
- Filter by People, Jobs, Companies, Posts after search
- Use search filters in left sidebar

## Profile Navigation
- Profile sections are collapsible cards
- Experience, Education, Skills sections
- "Connect" button to send connection request
- "Message" button for InMail

## Jobs
- Jobs tab in main navigation
- "Easy Apply" for quick applications
- Save jobs with bookmark icon
- Job alerts via bell icon

## Feed
- Home tab shows feed
- Like, Comment, Repost buttons under posts
- "Start a post" at top of feed

## Tips
- LinkedIn has aggressive rate limiting
- Wait between rapid actions
- Some features require login
